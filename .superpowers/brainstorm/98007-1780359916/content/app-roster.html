Which Apps Are In V1?

Select all you want in the first version. You can add more later — but smaller V1 = faster to ship.

🎮

Manual Control (stub exists)

Keyboard jog — press keys to move individual joints or in Cartesian XYZ. Live pose readout. Stop button.

🕹️

Xbox Controller (stub exists)

Bluetooth gamepad → Cartesian jog. Left stick = XY, right stick = Z + rotation. Bumpers = mode switch.

👁️

LLM Vision

Webcam feed + Claude/GPT-4V. Type or speak a command ("pick up the red block"), arm executes it. Live camera view in UI.

📊

Telemetry

Live dashboard — all 6 joint angles, end-effector XYZ, arm status, error state. No control, just monitoring.

⏺️

Sequence Recorder

Record a series of poses, then replay them in a loop. Like a teach pendant — move arm, hit record, save sequence, play it back.

📍

Pose Library

Name and save poses (home, rest, pick, place). One-tap to send arm to a saved pose. Simple JSON storage.

My recommendation for V1: Manual + Xbox + Telemetry — these wire up the existing stubs and give you a fully working system to build on. Add Vision + Sequences in V2 once motion is confirmed working.
